# ROTA
ROTA (Rastreamento, Organização e Transporte Ágil) - Aplicativo móvel para controle de transportes desenvolvido em Flutter/Dart.

Objetivo - Fornecer um aplicativo que auxilie na gestão de rotas de transporte, além da organização e controle de estoque, permitindo que uma transportadora ou empresa de logística cadastre sua frota, seus motoristas, seus produtos e suas cargas, e planeje rotas vinculando esses recursos, acompanhando tudo através de indicadores visuais.

Como executar:
flutter pub get
flutter run

GUIA PARA USAR NO VSCODE E MICROSOFT EDGE:

1 - Baixar a pasta zipada do ROTA, na versão mais recente
2 - Abra o VSCODE
3 - Tenha o flutter e dart instalados
4 - Digite flutter pub get
5 - Digite flutter run -d edge
6 - Espero um pouco, se tudo executar sem erros, o terminal deve abrir o edge em poucos minutos

COMANDOS ÚTEIS:

r - atualiza a página do edge com as mudanças já realizadas no código
R - reseta o app, útil para mudanças mais pesadas
q - mata o terminal
flutter clean - excelente para resetar, você limpa todas as dependências, daí , basta digitar flutter pub get novamente para baixá-las
